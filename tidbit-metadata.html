The Solution

Structured metadata

Added meta tags to every tidbit:

<meta name="tidbit-type" content="founder">
<meta name="tidbit-subject" content="Kavindi">
<meta name="tidbit-company" content="Foodie">

Now tidbits are machine-readable.
